# ReleasyConsumer

A component that injects `Relay` `environment` into any children.

## Usage

```javascript
import { ReleasyConsumer } from 'react-releasy';

// ... more code here

render() {
  <ReleasyConsumer>
    <MyComponent />
  </ReleasyConsumer>
}
```

## Props

- **children**: a `React` component
